How it works

Manual import steps (if auto fails)

  1. Click Download JSON backup above to save a .json file.
  2. Open your Project Tracker.
  3. Go to the ⚙ Settings tab.
  4. Under Data, click ↑ Import JSON backup.
  5. Select the downloaded .json file — all projects will load.
